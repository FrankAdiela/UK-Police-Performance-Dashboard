# UK Police Performance Dashboard

## Overview

This project demonstrates how Power BI can be used to analyse UK Police crime data and transform raw data into meaningful business insights.

The dashboard focuses on:

- Crime trends
- Crime categories
- Geographic analysis
- Police performance KPIs
- Interactive filtering
- Executive reporting

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
- GitHub
